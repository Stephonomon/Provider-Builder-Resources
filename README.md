## ⚠️ Important: CHOP-Specific Content

This guide was originally developed at the **Children's Hospital of Philadelphia (CHOP)**. Many references are institution-specific, including:

- Environment URLs
- Internal ticketing systems (ServiceNow)
- Contact names and email addresses
- Organizational names (DTS)
- Builder tier names ("Younglings," "Padawans," "Jedi Knights," "Jedi Masters")
- Naming convention prefixes (`CHOP`)

**If you are adopting this guide at another organization**, update the following before distributing:

- [ ] Environment and portal URLs
- [ ] Ticketing/service desk references
- [ ] Internal contact names and email addresses
- [ ] Organization prefix in naming conventions
- [ ] Builder tier names and governance body names
- [ ] Analytics/CDW access pathways

The core Epic build concepts, governance frameworks, and clinical decision support standards are broadly applicable across Epic organizations and should require only targeted edits.
